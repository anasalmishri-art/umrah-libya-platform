# 🕋 منصة عمرة ليبيا - Umrah Libya Platform

منصة ويب متكاملة لخدمات العمرة في السوق الليبي، تجمع شركات العمرة المعتمدة تحت مظلة واحدة.

## ✨ المميزات الرئيسية

### للأدمن (مدير المنصة)
- ✅ إدارة الشركات (الموافقة/الرفض/الإيقاف)
- ✅ إنشاء حسابات شركات يدوياً مع تعيين كلمة المرور
- ✅ إعادة تعيين كلمات المرور
- ✅ تأكيد الدفع وإدارة الطلبات
- ✅ الموافقة على الباقات المميزة والعروض
- ✅ لوحة CMS كاملة لتعديل محتوى الموقع
- ✅ إحصائيات شاملة وتقارير الإيرادات

### لشركات العمرة
- ✅ تسجيل ذاتي وانتظار الموافقة
- ✅ إضافة وتعديل وحذف الباقات
- ✅ طلب تمييز الباقات (مدفوع)
- ✅ **رفع PDF/صورة واستخراج الباقات بالذكاء الاصطناعي**
- ✅ إدارة الطلبات والتواصل مع العملاء عبر واتساب

### للعملاء
- ✅ تصفح الباقات والشركات
- ✅ حجز الباقات والتحويل لواتساب لإتمام الدفع
- ✅ لوحة حساب شخصي
- ✅ رسائل تلقائية (تأكيد الحجز، تعليمات الدفع، دليل العمرة، قائمة التجهيزات)

## 🛠️ التقنيات المستخدمة

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4 + shadcn/ui
- **Database**: Prisma ORM + SQLite (تدعم PostgreSQL للإنتاج)
- **Auth**: JWT + bcryptjs
- **AI**: z-ai-web-dev-sdk (VLM لقراءة الملفات)
- **State**: Zustand
- **Language**: عربية كاملة + RTL

## 📋 المتطلبات

- Node.js 18+ أو Bun
- قاعدة بيانات SQLite (للتطوير) أو PostgreSQL (للإنتاج)

## 🚀 التشغيل المحلي

```bash
# تثبيت الحزم
bun install

# إعداد قاعدة البيانات
bun run db:push

# تشغيل البيانات الأولية
bun run seed

# تشغيل التطبيق
bun run dev
```
