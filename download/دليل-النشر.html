🌟 الخيار الأول: النشر على Vercel (موصى به - مجاني)

Vercel هي منصة استضافة مجانية تتيح لك نشر التطبيق بسهولة وبدون تكلفة.

المميزات:
  • ✅ مجاني 100% (للمشاريع الصغيرة والمتوسطة)
  • ✅ سرعة عالية وتوزيع عالمي
  • ✅ شهادة SSL مجانية (HTTPS)
  • ✅ دومين فرعي مجاني (your-app.vercel.app)
  • ✅ إعادة نشر تلقائية عند كل تحديث

الخطوات:

1 إنشاء حساب على GitHub (إذا لم يكن لديك)
  1. اذهب إلى https://github.com/signup
  2. أدخل بريدك الإلكتروني وكلمة المرور
  3. اتبع التعليمات لإنشاء الحساب
2 رفع المشروع على GitHub
  • تواصل معي وسأقوم برفع المشروع لك على GitHub
  • أو اطلب من مبرمج رفع المجلد /home/z/my-project على GitHub
3 إنشاء حساب على Vercel
  1. اذهب إلى https://vercel.com/signup
  2. اضغط "Sign Up with GitHub" (التسجيل عبر GitHub)
  3. وافق على الصلاحيات المطلوبة
4 ربط قاعدة البيانات (Neon PostgreSQL - مجاني)
  1. اذهب إلى https://neon.tech
  2. أنشئ حساباً مجانياً
  3. أنشئ مشروعاً جديداً واحصل على رابط الاتصال (DATABASE_URL)
5 نشر التطبيق على Vercel
  1. في لوحة تحكم Vercel، اضغط "Add New Project"
  2. اختر مستودع GitHub الخاص بالمشروع
  3. في إعدادات Environment Variables، أضف:
    • DATABASE_URL = رابط PostgreSQL من Neon
    • JWT_SECRET = أي نص عشوائي طويل
    • NEXTAUTH_URL = https://your-app.vercel.app
  4. اضغط "Deploy" وانتظر دقائق
6 تهيئة قاعدة البيانات

بعد النشر، اذهب إلى تبويب "Settings" في Vercel:

  • ابحث عن "Commands" وأضف أمر: bun run db:push && bun run seed
  • أو شغّل الأمر من Vercel CLI بعد النشر
🎉 مبروك! تطبيقك متاح الآن على:
https://your-app-name.vercel.app

🌟 الخيار الثاني: النشر على خادم خاص (VPS)

إذا كنت تفضل خادماً خاصاً (أفضل للبيانات الحساسة):

المتطلبات:

خادم Linux (Ubuntu 22.04+) - يمكن استئجاره من:

  • DigitalOcean (https://digitalocean.com) - يبدأ من 5$/شهر
  • Hetzner (https://hetzner.com) - يبدأ من 4€/شهر
  • OVH (https://ovh.com) - يبدأ من 3€/شهر

الخطوات:

  1. استأجر خادماً بنظام Ubuntu
  2. ثبّت Node.js و Bun و Nginx
  3. ارفع ملفات المشروع
  4. شغّل bun install && bun run build
  5. استخدم PM2 لتشغيل التطبيق بشكل دائم
  6. ثبّت Nginx كـ Reverse Proxy

الدومين:

  • اشترِ دوماً من Namecheap أو GoDaddy
  • وجّه DNS إلى IP الخادم
  • ثبّت شهادة SSL مجانية عبر Let's Encrypt

🌟 الخيار الثالث: رابط المعاينة الفوري (متاح الآن)

حتى تنتهي من إجراءات النشر الكامل، يمكنك استخدام رابط المعاينة الذي يتيح لأي شخص الوصول للتطبيق:

رابط المعاينة: 
https://preview-<bot-id>.space-z.ai/

ملاحظات:

  • هذا الرابط متاح فوراً ويمكن مشاركته
  • مناسب للاختبار والعرض الأولي
  • للنشر الدائم، استخدم الخيار الأول أو الثاني

🔐 إعدادات الأمان للإنتاج

بعد النشر، قم بالخطوات التالية فوراً:

1. تغيير كلمات المرور الافتراضية

سجّل دخولك كمدير (admin@umrah.ly / admin123) ثم:

  • غيّر كلمة مرور الأدمن من خلال إنشاء حساب جديد
  • أو اطلب مني مساعدتك في تغييرها برمجياً

2. تحديث JWT_SECRET

في ملف .env أو إعدادات Vercel:

  • غيّر قيمة JWT_SECRET إلى نص عشوائي طويل
  • استخدم: https://randomkeygen.com لتوليد مفتاح قوي

3. تفعيل HTTPS

  • على Vercel: تلقائي ومجاني
  • على VPS: استخدم Let's Encrypt (مجاني)

📞 معلومات التواصل المهمة

حدّث معلومات التواصل من لوحة الأدمن:

  1. سجّل دخولك كمدير
  2. اذهب إلى تبويب "المحتوى"
  3. اختر قسم "التواصل"
  4. حدّث:
    • رقم الهاتف
    • رقم واتساب
    • البريد الإلكتروني
    • العنوان
    • ساعات العمل

💡 نصائح مهمة

1. النسخ الاحتياطي

  • احتفظ بنسخة من قاعدة البيانات أسبوعياً
  • على Vercel + Neon: النسخ الاحتياطي تلقائي
  • على VPS: استخدم cron job للنسخ الاحتياطي اليومي

2. مراقبة الأداء

  • استخدم Google Analytics لتتبع الزوار
  • راقب أداء التطبيق عبر Vercel Analytics (مجاني)

3. تحسين محركات البحث (SEO)

  • أضف وصف الموقع في لوحة الأدمن > المحتوى
  • اربط الموقع بـ Google Search Console
  • أنشئ خريطة موقع (sitemap.xml)

4. الدومين المخصص

بدلاً من your-app.vercel.app، يمكنك استخدام دومين مخصص:

  • اشترِ دوماً مثل umrah-libya.com أو omrah.ly
  • في Vercel: Settings > Domains > Add Domain
  • وجّه DNS من مزود الدومين إلى Vercel

❓ الأسئلة الشائعة

س: هل أحتاج خبرة برمجية للنشر؟

ج: لا، إذا استخدمت Vercel. فقط اتبع الخطوات أو اطلب المساعدة.

س: كم تكلفة النشر؟

ج:

  • Vercel + Neon: مجاني للمشاريع الصغيرة
  • دومين مخصص: ~10-15$ سنوياً
  • VPS (اختياري): ~5$ شهرياً

س: ماذا لو احتجت مساعدة؟

ج: تواصل معي وسأرشدك خطوة بخطوة.

س: هل بياناتي آمنة؟

ج: نعم، كلمات المرور مشفّرة، والاتصال عبر HTTPS، وقاعدة البيانات محمية.

س: هل يمكنني تعديل المحتوى بدون برمجة؟

ج: نعم! من لوحة الأدمن > تبويب "المحتوى" يمكنك تعديل كل شيء.
